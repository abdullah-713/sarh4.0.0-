# 17. المصطلحات (قاموس عربي–إنجليزي)

## 17.1 مصطلحات النظام

| العربية | الإنجليزية | الوصف |
|---------|-----------|-------|
| سهر الإتقان | SARH Al-Itqan | اسم النظام — نظام إدارة رأس المال البشري |
| بوابة الموظف | Employee Portal | واجهة الموظف PWA على `/app` |
| لوحة الإدارة | Admin Panel | لوحة التحكم الإدارية على `/admin` |
| غرفة القيادة | Command Room | مجموعة التنقل للتقارير المالية والخزنة السرية |
| المرآة الشخصية | Personal Mirror | مؤشرات الأداء الذاتية للموظف |
| ساعة الفرصة الضائعة | Lost Opportunity Clock | عداد الخسائر المالية اللحظي |
| الخزنة السرية | Whistleblower Vault | خزنة البلاغات السرية المشفّرة |

## 17.2 مصطلحات مالية

| العربية | الإنجليزية | الوصف |
|---------|-----------|-------|
| تكلفة الدقيقة | Cost Per Minute | قيمة الدقيقة الواحدة من الراتب |
| القيمة لكل دقيقة | VPM (Value Per Minute) | تكلفة الدقيقة على مستوى الفرع |
| مضاعف الخسارة | Loss Multiplier | مُعامل تضخيم تكلفة التأخير |
| الراتب الإجمالي | Total Salary | الأساسي + جميع البدلات |
| صافي الراتب | Net Salary | بعد الخصومات + الإضافات |
| خصم التأخير | Delay Deduction | الخصم المالي بسبب التأخير |
| أجر العمل الإضافي | Overtime Pay | تعويض العمل الإضافي (× 1.5) |
| كشف الراتب | Payroll | بيان الراتب الشهري |
| مصفوفة العائد | ROI Matrix | مقارنة عائد الاستثمار بين الفروع |
